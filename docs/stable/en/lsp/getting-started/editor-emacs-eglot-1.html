Configuring eglot can be as straightforward as updating the eglot-server-programs list to register the language server and how to start it, and running eglot-ensure whenever we open an *.rst file. 

(require 'eglot)
(add-to-list 'eglot-server-programs
             `(rst-mode . (,(executable-find "python3") "-m" "esbonio")))
(add-hook 'rst-mode-hook 'eglot-ensure)
