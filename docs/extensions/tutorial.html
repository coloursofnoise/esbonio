Tutorial¶

The tutorial extension can be enabled by including esbonio.tutorial in your extensions list

extensions = [
   "esbonio.tutorial",
]

Has the ability to write tutorials and export them as Jupyter notebooks. These can then made available online through cli commands or a notebook hosting service like Binder.
