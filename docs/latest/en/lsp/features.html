Features¶

This page contains a quick overview of the features offered by the Language Server

Completion¶

The language server implements textDocument/completion and can offer suggestions in a variety of contexts.

Document Symbols¶

The language server implements textDocument/documentSymbol which powers features like the “Outline” view in VSCode.

Diagnostics¶

Using textDocument/publishDiagnostics the language server is able to report Sphinx errors that are reported during builds.

Example diagnostic messages from Sphinx¶

Goto Definition¶

The language server implements textDocument/definition to provide the location of objects linked to by certain roles. Currently only the :ref: and :doc: roles are supported.
