get_location_type(doc, position)[ffynhonnell]¶

Given a document and a position, return the kind of location that represents.

This will return one of the following values:

  • rst: Indicates that the position is within an *.rst document

  • py: Indicates that the position is within code in a *.py document

  • docstring: Indicates that the position is within a docstring in a *.py document.

Paramedrau

  • doc (pygls.workspace.Document) -- The document associated with the given position

  • position (pygls.lsp.types.basic_structures.Position) -- The position to determine the type of.
