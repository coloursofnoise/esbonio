Directives¶

Logic around directive completions goes here.

esbonio.lsp.directives.DIRECTIVE = re.compile("\n    (\\s*)                           # directives can be indented\n    (?P<directive>\n      \\.\\.                          # directives start with a comment\n      [ ]?                          #, re.VERBOSE)¶

A regular expression to detect and parse partial and complete directives.

This does not include any options or content that may be included underneath the initial declaration. The language server breaks a directive down into a number of parts:

                vvvvvv argument
.. c:function:: malloc
^^^^^^^^^^^^^^^ directive
     ^^^^^^^^ name
   ^ domain (optional)
esbonio.lsp.directives.DIRECTIVE_OPTION = re.compile("\n    (?P<indent>\\s+)       # directive options must be indented\n    (?P<option>\n      :                   # options start with a ':'\n      (?P<name>[\\w-]+)?   # options have a name\n      :?   , re.VERBOSE)¶

A regular expression used to detect and parse partial and complete directive options.

The language server breaks an option down into a number of parts:

            vvvvvv value
|   :align: center
    ^^^^^^^ option
     ^^^^^ name
 ^^^ indent
